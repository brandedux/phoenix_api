# Phoenix 1.4 Oauth2 API
[![Build Status](https://api.travis-ci.org/brandedux/phoenix_api.svg?branch=master)](https://travis-ci.org/brandedux/phoenix_api)

Create API Tokens in your Phoenix 1.4 app.

#### Working Features:
<b>Authentication</b>
- [x] Register
- [x] Sign in
- [x] Sign out
- [ ] Password reminder (Email)
- [ ] Hide / Show links based on authentication state
- [ ] Error Handling

<b>User Profile</b>
- [x] Edit Email
- [x] Edit Password
- [ ] Upload Avatar

<b>Data List</b>
- [ ] Websockets
- [ ] Render Data List
- [ ] Detail View
- [ ] Data Visualizations (Chart.js, Three.js)
